# @spikequiz/shared

Shared types, interfaces, and constants used across all packages.

## Usage

```ts
import { Question, Subject, SUBJECTS } from '@spikequiz/shared'
import type { PrimaryClassifierInput } from '@spikequiz/shared/types'
```

## Exports

- `types.ts` - All TypeScript interfaces and types
- `constants.ts` - Constant values and enums
- `index.ts` - Re-exports everything
